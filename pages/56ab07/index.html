    Hexo

    # 自动生成permalink

    1. 添加插件,Github地址 (opens new window)
    npm install hexo-abbrlink --save
    1. 修改配置文件_config.yml
    permalink: pages/:abbrlink/
abbrlink:
    alg: crc32   #算法： crc16(default) and crc32
    rep: hex     #进制： dec(default) and hex
    1. 清除缓存重新生成
    hexo clean
hexo g
